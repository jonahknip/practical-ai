## Website

The website is built with React + TypeScript + Vite.

**Live site:** https://predeploy-0ee90a96-practicalai-bg3sg6vi.manus.space/

### Local Development
```bash
cd website
npm install
npm run dev
```

### Build for Production
```bash
npm run build
```
