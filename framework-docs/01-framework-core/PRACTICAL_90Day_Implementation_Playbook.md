## TROUBLESHOOTING GUIDE

### Low Adoption (<60% by Week 3)

**Diagnose:**
- Is training inadequate?
- Is solution too complex?
- Is value unclear?
- Are there blockers (access, time)?
- Is change resistance high?

**Fix:**
- More hands-on training
- Simplify interface
- Show tangible time savings
- Remove access barriers
- Address fears directly

---

### Time Savings Not Materializing

**Diagnose:**
- Is solution being used correctly?
- Are people using it for right tasks?
- Is learning curve longer than expected?
- Are results quality issues slowing down?

**Fix:**
- Observe actual usage
- Provide better examples
- Create templates for common use cases
- Improve quality of outputs

---

### Quality Issues

**Diagnose:**
- Are prompts too generic?
- Is context missing?
- Are users skipping review step?
- Is AI hallucinating facts?

**Fix:**
- Refine prompts with specificity
- Add context requirements
- Emphasize human review
- Build fact-checking into process

---

### Champion Burnout

**Diagnose:**
- Taking too much time?
- Not getting recognition?
- Feeling unsupported?
- Overwhelmed by questions?

**Fix:**
- Reduce their load
- Public recognition
- More resources/support
- Recruit additional champions
